# Hello Ahmed Webapp

A minimal static web page that displays "Hello Ahmed" on the home page.

## Run locally

- Open `index.html` directly in your browser, or
- Serve the folder with a simple local server:

```bash
# macOS (Python 3)
cd /Users/hnceloz/Desktop/CURSOR
python3 -m http.server 8080
# then visit http://localhost:8080/
```

## Deploy

You can host these static files on any static hosting (GitHub Pages, Netlify, Vercel, etc.).
